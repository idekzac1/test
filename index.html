รายงานวิชาเว็บไซต์

การพัฒนาและการทำงานของเว็บไซต์

บทนำ

เว็บไซต์เป็นเทคโนโลยีที่สำคัญในยุคดิจิทัล ใช้สำหรับเผยแพร่ข้อมูล การทำธุรกิจ และการสื่อสารผ่านอินเทอร์เน็ต ปัจจุบันองค์กรและบุคคลทั่วไปสามารถสร้างเว็บไซต์ เพื่อแสดงข้อมูลหรือให้บริการออนไลน์ได้

ความรู้พื้นฐานเกี่ยวกับเว็บไซต์

Website

เว็บไซต์คือชุดของหน้าเว็บที่เชื่อมโยงกันและสามารถเข้าถึงผ่านอินเทอร์เน็ต

Web Page

หน้าเว็บเพจคือเอกสาร HTML ที่แสดงผลผ่านเว็บเบราว์เซอร์

Web Browser

โปรแกรมที่ใช้เปิดเว็บไซต์ เช่น Chrome, Edge, Firefox

องค์ประกอบของเว็บไซต์

Domain

ชื่อเว็บไซต์ เช่น example.com

Hosting

พื้นที่เก็บไฟล์เว็บไซต์บนเซิร์ฟเวอร์

Database

ระบบเก็บข้อมูล เช่น MySQL

Server

เครื่องที่ให้บริการเว็บไซต์

เทคโนโลยีที่ใช้พัฒนาเว็บไซต์

HTML

โครงสร้างของหน้าเว็บ

CSS

ใช้ตกแต่งและออกแบบหน้าเว็บ

JavaScript

ทำให้เว็บมีการโต้ตอบ

PHP

ภาษา backend สำหรับเว็บไซต์

  4. ทดสอบระบบ
  5. เผยแพร่เว็บไซต์
